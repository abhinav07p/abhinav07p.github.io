üéØ Inventory Optimization

Input: Forecast demand + current inventory levels + cost parameters

Optimization Model (Pyomo):

# Minimize: holding_cost + ordering_cost + stockout_cost # Subject to: # - Inventory balance constraints # - Service level >= 95% # - Order quantities >= MOQ model = ConcreteModel() model.reorder_point = Var(within=NonNegativeReals) model.order_qty = Var(within=NonNegativeReals) def total_cost_rule(m): return holding * m.reorder_point + ordering * m.order_qty solver = SolverFactory('highs') results = solver.solve(model)

Output: Optimal reorder points, safety stock, order quantities ‚Üí INVENTORY_POLICIES table

‚Üì
5

üé≤ Monte Carlo Simulation

Input: Optimal inventory policies + demand uncertainty

Simulation: Run 10,000 scenarios with demand drawn from forecast distribution

Metrics Calculated:

  • Expected total cost
  • Stockout probability
  • Value-at-Risk (VaR) at 95%
  • Conditional Value-at-Risk (CVaR)

Output: Risk-adjusted policy recommendations ‚Üí SIMULATION_RESULTS table
